# BUILDAPP

This is a **practice project** built to learn how to design and implement a modern **AI-powered SaaS application**.

## What This Project Covers

### 1. Building a SaaS Application
- Pricing system
- Usage tracking
- Rate limiting

### 2. Building an AI Agent System
- Using **Inngest Agents**
- Using **E2B** for sandboxed code execution
- Using **Google Gemini** for AI capabilities

### 3. Authentication
- **Clerk** for user authentication and session management

### 4. Database
- **Dockerized PostgreSQL** for local and consistent database environments

### 5. ORM
- **Prisma ORM** for database schema management and type-safe queries

### 6. Properly Structured Codebase
- Organizing code for readability, maintainability, and scalability

### 7. Server Actions
- Using **Next.js Server Actions** for handling forms and server-side logic

### 8. Proper Error Handling
- Handling application and API errors in a clean and maintainable way
